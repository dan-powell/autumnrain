# Agent Instructions

This project uses Lando for its development environment. All PHP, Composer, and NPM commands **must** be run via Lando.

To execute these commands, prefix them with `lando`:

*   **PHP commands:** `lando php <command>` (e.g., `lando php artisan migrate`)
*   **Composer commands:** `lando composer <command>` (e.g., `lando composer install`)
*   **NPM commands:** `lando npm <command>` (e.g., `lando npm run dev`)
